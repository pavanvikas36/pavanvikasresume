URL

A URL (Uniform Resource Locator) is essentially the address of a resource on the internet. Think of it as a digital street address that directs your browser to a specific webpage, image, video, or other online content.

Anatomy of a URL

A URL is composed of several parts.

  1. Protocol
  2. Domain Name
  3. Path

Protocol

A set of rules that govern how data is transmitted between computers over a network.

  1. Internet Protocol
  2. Network Protocol

Domain name

A domain is a unique, human-readable identifier that represents one or more IP addresses on the internet.

  1. .com
  2. .net
  3. org

Path

the path is the component that specifies the location of a specific resource within a web server's file system.

  1. Simple path to single page
  2. Path with multiple directories
  3. Path with a File name

SDLC

The Software Development Life Cycle (SDLC) is a conceptual framework that defines all the steps involved in the development of software applications, from the initial planning and requirements gathering phase to deployment, maintenance, and eventual retirement.

  1. Planning/Requirements
  2. Design
  3. Implementation/Development
  4. Testing
  5. Deployment
  6. Maintenance

Common SDLC Models:

  1. Waterfall
  2. Agile
  3. Spiral
